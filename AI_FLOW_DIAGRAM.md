## Data Flow

### Request Data Structure
```typescript
{
  task: {
    id: string
    title: string
    description: string
    status: "todo" | "in-progress" | "completed"
    priority: "low" | "medium" | "high"
    dueDate: string (ISO date)
    course: string
    subtasks?: string[]
  },
  allTasks: Task[] // All user's tasks for context
}
```

### AI Prompt Structure
```
SYSTEM PROMPT (defines AI's role)
â†“
USER PROMPT (provides context):
  - Task details
  - Learner's workload
  - Related tasks
  - Specific requests
â†“
AI RESPONSE (structured JSON):
  {
    subtasks: string[]
    schedule: string
    strategy: string
    resources: string
  }
```

### Response Data Structure
```typescript
{
  success: boolean
  suggestions: [
    {
      type: "subtasks" | "schedule" | "strategy" | "resources"
      title: string
      content: string
      subtasks?: string[] // Only for subtasks type
    }
  ],
  rawResponse?: object // Original AI response
}
```
